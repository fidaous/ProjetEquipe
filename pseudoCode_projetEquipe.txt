
AFFICHAGE PRINCIPAL:

Menu utilisateur:

1- Se connecter à votre compte
2- Se connecter comme administrateur
3- Quitter



Si l'utilisateur choisi 1:

- Afficher message "Veuillez entrer vos identifiants"

Si l'utilisateur entre des infos enregistrées, afficher le menu du compte personnel.

Sinon, afficher message d'erreur. (Ne pas spécifier l'erreur)

Si l'utilisateur fait 3 erreurs, bloquer le compte.

Afficher message "Votre compte est bloqué".



Si l'utilisateur choisi 2:

- Afficher message "Veuillez entrer les identifiants administrateur"

Si l'utilisateur entre des infos enregistriées, afficher le menu administrateur.

Sinon, afficher message d'erreur. (Ne pas spécifier l'erreur)

Si l'utilisateur fait 3 erreurs, le guichet tombe en mode panne.



Si l'utilisateur choisi 3:

Le système se termine.




AFFICHAGE MENU COMPTE PERSONNEL:

1- Changer le mot de passe
2- Déposer un montant dans un compte
3- Retirer un montant d'un compte
4- Afficher le solde du compte (Chèque ou épargne)
5- Effectuer un virement entre les comptes
6- Payer une facture
7- Fermer la session


Si l'utilisateur choisi 1:

Afficher message "Changer de mot de passe"

Si le nouveau mot de passe respecte les règles

Afficher message "Confirmer le nouveau mot de passe"

Lorsque l'utilisateur confirme le nouveau mot de passe, le système retourne au menu usager.




Si l'utilisateur choisi l'option 2 du menu compte perso:

Afficher message "Déposer un montant dans un compte"

Afficher message "Choisissez un compte: 1- Chèque 2- Épargne"

Si l'utilisateur choisit Chèque, le dépôt va se faire dans le compte chèque.

Si l'utilisateur choisit Épargne, le dépôt va se faire dans le compte épargne.

L'utilisateur doit entrer le montant à déposer.

Tant que le montant à déposer est négatif, afficher message "Veuillez entrer un montant valide"

Lorsque le dépôt est effectué, afficher le solde du compte et retour au menu du compte.




Si l'utilisateur choisi l'option 4 du menu compte perso:

Afficher message "Veuillez sélectionner un compte (Chèque ou Épargne)"

Si l'utilisateur sélectionne Chèque, afficher le solde du compte Chèque.

Si l'utilisateur sélectionne Épargne, afficher le solde du compte Épargne.

(Le montant doit être affiché dans le format xx xxx.xx$ )
